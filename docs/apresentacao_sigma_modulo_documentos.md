# Gestão e Personalização de Documentos - Guia do Usuário

O Sistema Sigma oferece um poderoso módulo para criação, personalização e emissão de documentos oficiais de sua Loja. Este guia apresenta as funcionalidades disponíveis e como utilizá-las para elevar o padrão da secretaria de sua Loja.

## 1. Visão Geral
Com o Sigma, sua Loja pode emitir:
- **Balaústres (Atas)**: Com geração automática de texto baseada nos registros da sessão e editor rico para ajustes finos.
- **Pranchas de Convocação (Editais)**: Formatados automaticamente com a Ordem do Dia e dados da sessão.
- **Certificados de Presença**: Para visitantes, com validação digital via QR Code.

## 2. Personalização da Identidade Visual
Cada Loja é única. O Sigma permite que você configure:
- **Logo da Loja**: Seu brasão aparecerá automaticamente no cabeçalho de todos os documentos.
- **Cabeçalhos Personalizados**: Escolha entre diversos estilos (Clássico, Moderno, Minimalista) para adequar o documento à formalidade exigida.
- **Fontes e Margens**: Ajuste o tamanho da fonte e o espaçamento para garantir a melhor leitura e economia de papel.

## 3. Segurança e Validade Jurídica
Todos os documentos emitidos pelo sistema possuem mecanismos avançados de autenticidade:
- **QR Code de Validação**: Qualquer pessoa com acesso ao documento impresso pode escanear o código para verificar se ele é autêntico e foi emitido pelo sistema oficial.
- **Assinatura Digital**: O sistema registra quem emitiu o documento (Secretário ou Venerável), data e hora, garantindo rastreabilidade.

## 4. Fluxo de Trabalho Simplificado

### 4.1. Emitindo um Balaústre
1. **Realize a Sessão**: Lance as presenças e cargos no sistema como de costume.
2. **Gere o Rascunho**: O sistema criará um texto base completo ("Aos XX dias do mês de..."), economizando horas de digitação.
3. **Edite e Refine**: Use nosso editor de texto para adicionar detalhes específicos das discussões em Loja.
4. **Salve e Assine**: Com um clique, gere o PDF final com todas as assinaturas e o QR Code.

### 4.2. Emitindo Certificados para Visitantes
1. Vá até a lista de presença da sessão.
2. Selecione o visitante desejado.
3. Clique em "Gerar Certificado".
4. O sistema gera um PDF pronto para impressão ou envio por WhatsApp/Email, com código de validação único.

## 5. Dúvidas Frequentes

**Posso usar meu próprio modelo de Word?**
O Sigma utiliza modelos inteligentes (Templates) que se adaptam aos dados. Você não precisa mais de arquivos Word soltos. Configure seu estilo no painel de configurações e o sistema aplicará a todos os documentos.

**A assinatura digital substitui o cartório?**
A validação via QR Code garante autenticidade dentro do ecossistema Sigma e para conferência entre Lojas. Para validade jurídica externa (cartórios), o sistema está preparado para integração futura com certificados e-CNPJ/e-CPF.
